# Life in D
Port of [Life game](https://github.com/contextfreecode/life) to D and Parin engine.

## Building
To install:

```sh
dub run parin:setup
```

To run native version:

```sh
dub run --build=release
```

To run web version:

```sh
dub run parin:web --build=release
```
